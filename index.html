Calicis Calculator

The calculator GUI in Java Swing features a clean and intuitive design for performing basic arithmetic operations. The main interface is built using a JFrame as the container, with a non-editable JTextField at the top to display inputs and results. Below the display, a JPanel arranges the calculator buttons in a grid layout, including buttons for digits (0-9), basic arithmetic operators (+, -, *, /), a clear button (C), and an equals button (=). Each button is implemented with a JButton, and action listeners are attached to handle the respective operations. The layout ensures proper alignment and responsiveness, making the calculator easy to use.

Rock Paper Scissors Game

The Rock-Paper-Scissors game implemented in Java Swing features a simple and interactive graphical user interface. The main window is built using a JFrame containing a JPanel to organize components. At the top, a JLabel displays instructions or game status messages, such as "Choose your move!" or "You win!" Below the label, three JButton elements represent the player's options: "Rock," "Paper," and "Scissors." These buttons are equipped with action listeners to capture the player's choice. When a button is clicked, the game logic randomly selects the computer's move and determines the winner based on standard Rock-Paper-Scissors rules. The results are displayed in another JLabel below the buttons, showing both choices and the outcome (e.g., "You chose Rock. Computer chose Scissors. You win!"). The interface is enhanced with a "Play Again" button that resets the game state, allowing for repeated rounds without restarting the application. The layout is arranged using a combination of BorderLayout and FlowLayout, ensuring an organized and visually appealing design.

TrueJans Inventory Management System

The TrueJans Inventory Management System is a desktop application developed using VB.NET and an MS Access database to streamline inventory tracking and management. The system features a user-friendly interface designed with clear navigation panels and responsive controls to ensure ease of use for staff and administrators. The main interface includes modules for product management, stock monitoring, sales tracking, and report generation. The product management module allows users to add, update, or delete product information such as name, category, price, and stock quantity. The stock monitoring module provides real-time updates on inventory levels, with alerts for low-stock items. The sales tracking module records transactions, linking them to inventory updates to ensure data consistency. Finally, the report generation module enables the creation of detailed reports, including sales summaries, stock histories, and low-stock alerts, which can be exported for further analysis. The application connects to an MS Access database, ensuring data is securely stored and easily retrievable. Database operations, such as inserting, updating, and querying data, are implemented using ADO.NET. User authentication is integrated to protect sensitive data, and role-based access control ensures that only authorized users can perform certain actions, such as modifying product details or accessing reports. Overall, the system enhances efficiency, minimizes errors, and provides comprehensive inventory management for small to medium-sized businesses.
